Create a virtual environment: 
    1. python -m venv myenv (Python module in myenv)
    2. conda create -p myenv python==3.12 -y(Create it in the current folder with the name myenv)
Activate the environment: myenv\Scripts\activate

Now we can install our requirements like python, numpy, pandas versions etc

Check the python version: python --version(3.12.0)

Created a requirements.txt file

Install the required packages: pip install fastapi uvicorn pydantic
